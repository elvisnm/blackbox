Blackbox

A development framework for AI-assisted teams.

Everyone uses AI. Without a shared path, AI conversations drift apart. Blackbox makes them converge by connecting POs, Designers, and Devs through structured artifacts.

Get Started See the Workflow

The Problem

Teams use AI in isolation. The PO, Designer, and Developer each talk to their own AI separately with no shared artifact.

  • Inconsistent output across roles
  • Key architectural decisions lost in private chats
  • Repeated context sharing leading to friction

The Solution

Two connected artifacts enforced by Claude AI: Asana for workflow, and Blueprints for the recipe.

  • Claude acts as a quality gate on both sides
  • PO's Claude writes highly structured tickets
  • Dev's Claude generates technical blueprints from tickets

How It Works

A seamless handover process where AI maintains the context from planning to pull request.

1. Plan

PO creates structured ticket

2. Enrich

Designer adds context (optional)

3. Scaffold

Dev generates blueprint

4. Refine

AI researches codebase

5. Ship

Implements & opens PR

Stay in your flow

Nobody learns a new tool. Each role operates in the environment they already know.

PO & Designer

Work entirely in Asana. No git, terminals, or repositories needed.

Developer

Work entirely in the Code Editor via the Blackbox CLI tools.

QA

Test the final implementation strictly against the structured ticket.

Framework Architecture

Blueprint Template

The standardized Markdown structure that defines the feature, scope, technical plan, and testing strategy living alongside the code.

12 CLI Skills

A suite of command-line tools to scaffold, refine, review, and wrap-up blueprints, connecting local context with Claude's API.

Role Templates

Specific CLAUDE.md guidelines and prompt structures optimized for POs and Designers to generate perfect inputs.

Dashboard

A lightweight web application that aggregates and indexes blueprints across all your projects for persistent knowledge retrieval.

Developer Experience

blackbox-cli
/scaffold Generate initial blueprint from Asana URL
/refine AI researches local codebase to update plan
/implement Execute the technical steps defined in blueprint
/wrap-up Final quality check against initial requirements
/create-pr Generate branch and formatted PR description
/send-back Return ticket to PO or Designer via Asana
